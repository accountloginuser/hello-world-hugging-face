Hello World Hugging Face

Hugging Face is a platform that allows playground for testing machine learning in artificial intelligence.
This demo makes use of its feature of:
- Frontend space: Input from HTML form
- Backend space: Process input and respond back to frontend space with output


Backend space

A tutorial from Hugging Face for sentiment analysis will be referenced for testing the input text
Testing cycle: start with a relevant base model; tune model; test input with tuned model; return results;
On Hugging Face, create space with Docker blank template.
Backup the files already in the space (e.g. .gitattributes ; Dockerfile ; README.md ; app.py ; requirements.txt ;)
Then copy the files from backend folder to the backend space
Select App to build/serve and select the log to see its status


Frontend space

On Hugging Face, create space with Static blank template.
Backup the files already in the space (e.g. .gitattributes ; README.md ; index.html ; style.css ;)
Then copy the files from frontend folder to the frontend space
Configure the endpoint for backend space in file index.html
Select App to build/run the HTML form
To debug, can try view browser console.
